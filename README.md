# React Dashboard

A simple and interactive **React Dashboard UI** built with React, Material UI, and Recharts.
This project demonstrates dashboard layout design, charts, and UI components commonly used in admin panels.

---

## 🚀 Features

* 📊 Dashboard analytics UI
* 📈 Charts using Recharts
* 🎨 Material UI components
* ⚡ Responsive layout
* 🗺️ Google Maps integration
* 📱 Mobile friendly design

---

## 🛠️ Technologies Used

* React JS
* Material UI
* Recharts
* Google Map React
* CSS

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/react-dashboard.git
```

Go to the project folder:

```bash
cd react-dashboard
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Locally

Start the development server:

```bash
npm start
```

Then open your browser and go to:

```
http://localhost:3000
```

---

## ⚠️ Node Version Fix (Important)

If you get this error:

```
Error: error:0308010C:digital envelope routines::unsupported
```

Run this command before starting the project:

```bash
set NODE_OPTIONS=--openssl-legacy-provider
npm start
```

Or use **Node.js v18 LTS** for best compatibility.
